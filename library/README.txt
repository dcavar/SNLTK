INSTALLATION

Install the library files into the PLT-Scheme distribution (4.x) by:

On Windows:
- Running (double-click) the batch file plt-install.bat

On Unix (e.g. Linux, Mac OS X):
- Running the shell script plt-install.sh in a command line tool for example with:
. ./plt-install.sh

in the folder with the library files.

You can edit the files to adopt your individual PATH to the necessary executables.
